INSTRUCCIONES PARA INSTALAR LA FUENTE ENCODE SANS
==================================================

1. Descarga la fuente desde Google Fonts:
   https://fonts.google.com/specimen/Encode+Sans

2. Extrae el archivo ZIP descargado.

3. Copia los siguientes archivos .ttf o .woff2 en esta misma carpeta:
   - EncodeSans-Light.woff2      (peso 300)
   - EncodeSans-Regular.woff2   (peso 400)
   - EncodeSans-Medium.woff2    (peso 500)
   - EncodeSans-SemiBold.woff2  (peso 600)
   - EncodeSans-Bold.woff2      (peso 700)

   (Si descargas los .ttf, renómbralos igual pero con extensión .ttf)

4. El archivo variables.css ya tiene el @font-face configurado para ambas extensiones.
   No necesitas modificar ningún CSS — solo coloca los archivos aquí.

Descarga directa (ZIP completo):
https://fonts.gstatic.com/s/encodesans/v19/LDI_apOFNx6aLQqPY2se0H3OCk6DBGdyrA.zip
