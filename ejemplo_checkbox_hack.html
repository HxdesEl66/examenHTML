═══════════════════════════════════════════════════════════════

PREGUNTA 1: "¿Cómo ocultas un elemento pero sigues usarlo?"
RESPUESTA: 
  .elemento { display: none; }
  o
  .elemento { opacity: 0; position: absolute; }

═══════════════════════════════════════════════════════════════

PREGUNTA 2: "¿Cómo cambias el estilo cuando un checkbox está marcado?"
RESPUESTA:
  input:checked { ... }
  input:checked ~ .hermano { ... }

═══════════════════════════════════════════════════════════════

PREGUNTA 3: "¿Qué es el selector ~ (tilde)?"
RESPUESTA:
  Es el "selector de hermano". Afecta a elementos que vienen
  DESPUÉS del elemento seleccionado, en el mismo nivel.
  
  Ejemplo:
  
  
  
texto

  
  #x:checked ~ p { color: red; }
